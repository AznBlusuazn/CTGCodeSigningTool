ABOUT THIS APP

This app was created to reduce the number of
steps to apply a code signing certificate to
a .exe, .dll, .ocx, etc. file.

This application does require the official
Microsoft SignTool.exe app that is included
in the Windows 10 SDK from Microsoft.

https://docs.microsoft.com/en-us/windows/win32/seccrypto/signtool
